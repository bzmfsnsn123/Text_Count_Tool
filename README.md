# Text-Count-Tool
Python入门文本统计工具，可快速分析文本字数信息

## 功能介绍
1. 统计全部字符总数（包含空格、标点）
2. 统计剔除空格后的字符数量
3. 自动区分统计中文汉字、英文单词数量
4. 空输入校验，防止无效内容运行报错

## 运行环境
Python3，仅使用内置标准库，无需额外安装第三方模块

## 运行方法
1. 运行 text_count.py
2. 在控制台输入需要统计的文字
3. 一键查看完整统计数据

## 核心知识点
1. 字符串内置方法：len()、strip()、replace()、split()
2. 字符编码区间判断汉字
3. for循环遍历、if条件判断
4. 基础输入合法性校验
